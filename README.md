# Memory - AI Voice Companion

A complete hardware + software voice companion built for elderly users, developed from the ground up.

## About the Project

**Memory** is a practical AI voice companion designed to support elderly users with daily routines, medication reminders, and natural conversation.

It combines embedded hardware (ESP32) with real-time cloud AI (Gemini Live via AnveVoice) to create a warm, patient, and reliable daily companion.

### Key Features

- Real-time bidirectional voice streaming with I2S audio
- Persistent medication reminder system with natural dialogue flow
- Push-to-Talk (PTT) interface
- Custom Python bridge for low-latency audio streaming
- Echo handling and acoustic management
- Warm, elderly-friendly tone using Zephyr voice

### Technical Stack

- **Hardware**: ESP32 + I2S MEMS microphone + speaker
- **Firmware**: C++ with I2S audio handling and PTT
- **Bridge**: Python (asyncio + WebSockets)
- **AI**: Gemini Live 2.5 Flash (native audio) via AnveVoice

### Current Status

Working prototype with stable audio streaming, medication reminders, and basic echo mitigation.  
Actively improving sentence completion, response reliability, and conversation flow.
